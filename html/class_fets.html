Detailed Description

FETモジュール操作クラス

FETモジュールを操作する機能の抽象クラス
send() , recieve() が純粋仮想関数である
このクラス内の公開メソッドが主機能すべてである

Note
すべての公開メソッドはそのまま通信を行うので割り込みなどには注意
すべての通信情報は 4byte である
シリアル通信115200[bps]で制御周期が10[ms]のとき， 36メソッド/回未満にするべきである
Remarks
拡張クラスでデータ送受信を実装する必要がある
Attention
このクラスは抽象クラスであり，インスタンス化できない
派生クラスをインスタンス化する

ポート番号を指定せず実体化することで，そのクラス実体はモジュールとしてポートの指定は各メソッドで行う
ただしポート番号を指定して実体化しても，メソッドでポート指定されたときはそちらが優先される
出力ポートを None，入力ポートのみを指定しての実体化をしようとするとモジュールとして実体化される
Parameters
_idモジュールのID
デフォルトは 0x90
outputPort使用する出力ポートの番号 Fets::Out1 ~ Fets::Out7
指定しなくても良い
inputPort使用する入力ポートの番号 Fets::In1 ~ Fets::In7
指定しなくても良い
Note
引数はすべてを指定しなくても良い
すべて指定しなければ、ID:0x90 となり，ポート指定を各メソッドで行うようになる
なおいずれの実体でもIDが同じなら getInputState() , getOutputState() で得られるポート情報は同じになる
Attention
モジュールとしての実体を複数作ったり，ポート指定での実体を複数作ることは可能だが
モジュール実体とポート実体を混合させるのは不可能
モード干渉としてすべての公開メソッドで -1 を返す(はず)
