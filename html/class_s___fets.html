Detailed Description

FETモジュール操作機能の GR-SAKURA 実装用クラス

Fets クラスを継承した拡張クラス
Fets クラスのパブリックメンバをそのまま呼び出せる

Note
GR-SAKURA のシリアル通信 HardwareSerial を使用して send() , recieve() メソッドをオーバーライドしている
