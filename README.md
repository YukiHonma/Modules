## マスターとの通信/複数モジュール
 マスター対モジュールでの通信においてマルチスレーブ化が可能である．  
 ただし，Uartシリアル通信を用いる場合，モジュールからマスターへの通信ができるのは1モジュールのみである．  
 モジュール側のDIPスイッチと内部ファイルのIDを設定する必要がある．  

 また種類の違う複数のモジュールでもマルチスレーブ化ができるようにする．
 注意事項は上と同じ
